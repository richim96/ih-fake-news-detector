# Fake News Classifier

An NLP pipeline that classifies news headlines as **real (1)** or **fake (0)** using 5 feature extraction methods × 4 classifiers = 20 evaluated combinations.

---

## Dataset

| Split | Rows | Source |
|-------|------|--------|
| Train | 34,152 | `data/raw/training_data.csv` |
| Test  | 9,984  | `data/raw/testing_data.csv`  |

Tab-separated, no header. Columns: `is_real` (0/1/2) · `text` (headline).  
Class balance: ~51.5 % fake / 48.5 % real — effectively balanced.

---

## Pipeline

### 1 · Text Preprocessing

| Step | Detail |
|------|--------|
| Lowercase | |
| Mojibake fix | `‚` → `'`, `‚î` → ` - `, `√©` → `e`, etc. |
| Contraction expansion | `contractions.fix()` after apostrophe normalisation |
| HTML / URL removal | regex |
| Punctuation removal | keep `[a-z0-9\s]` only |
| Stopword removal | NLTK English stopwords |
| Lemmatisation | `WordNetLemmatizer` — preserves valid words vs. stemming |

### 2 · Feature Extraction

| # | Method | Dimensionality | Notes |
|---|--------|---------------|-------|
| 1 | Bag of Words | 10,000 | `CountVectorizer`, sparse |
| 2 | TF-IDF | 10,000 | `TfidfVectorizer`, sparse |
| 3 | Word2Vec | 100 | Trained on corpus; doc vec = mean of word vecs |
| 4 | fastText | 300 | Pretrained `cc.en.300.bin`; subword info handles OOV |
| 5 | BERT | 768 | `bert-base-uncased`, frozen CLS token |

### 3 · Classifiers

- **Logistic Regression** — linear baseline (`max_iter=1000`)
- **Random Forest** — 200 trees, `n_jobs=-1`
- **Naive Bayes** — MultinomialNB for sparse (BoW/TF-IDF), GaussianNB for dense
- **LightGBM** — gradient boosted trees, 200 estimators

All experiments use `random_state=96`. Feature transformers are fit on training data only (no leakage).
